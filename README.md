# API de Controle de Dieta Diária

## Conheça o projeto

Este projeto consiste no desenvolvimento de uma API RESTful para controle de dieta diária utilizando Flask.

A aplicação permitirá que usuários registrem suas refeições, informando nome, descrição, data e hora, além de um indicador que define se a refeição está dentro da dieta. O sistema oferecerá funcionalidades completas de CRUD (criar, ler, atualizar e deletar) para o gerenciamento das refeições, com persistência em banco de dados.

O objetivo do projeto é aplicar, de forma prática, conceitos avançados de Flask, proporcionando uma experiência real no desenvolvimento de APIs RESTful, com estruturação autônoma de rotas, propriedades e organização do código.

---

## Objetivo

Criar uma API funcional que permita o controle e acompanhamento da alimentação diária, possibilitando o registro detalhado das refeições e o gerenciamento completo desses dados.

Por se tratar de um desafio de nível avançado, espera-se autonomia na definição da estrutura da aplicação, incluindo rotas, regras de negócio e modelagem dos dados.

---

## Funcionalidades

A API deve permitir:

- Registrar refeições com nome, descrição, data/hora e indicador de dieta  
- Editar todos os dados de uma refeição existente  
- Excluir uma refeição  
- Listar todas as refeições de um usuário  
- Visualizar os detalhes de uma refeição específica  
- Persistir todas as informações em banco de dados  

---

## Tecnologias

- Python  
- Flask  
- Banco de dados (à escolha do desenvolvedor)  

---

## Requisitos do projeto

- Implementar uma API RESTful utilizando Flask  
- Estruturar rotas e propriedades de forma autônoma  
- Garantir organização e clareza no código  
- Utilizar persistência em banco de dados  
- Entregar um sistema funcional e coerente com os objetivos propostos

## Observações

Este projeto tem como foco o desenvolvimento prático e a consolidação de conhecimentos em Flask e arquitetura de APIs. As decisões de implementação fazem parte do desafio e devem refletir boas práticas de desenvolvimento.
